## 📌 Overview

**Smart Waste Segregation RL Environment** is an AI-powered simulation project designed to demonstrate how reinforcement learning and intelligent agents can be used for automated waste classification.

The system creates a virtual environment where an AI agent (LLM-based) interacts with different waste objects, makes classification decisions, and receives rewards based on accuracy.

This project aims to explore how modern AI techniques can improve smart waste management systems and support future real-world environmental solutions.

---

## ✨ Key Features

### 🧠 Reinforcement Learning Environment
- Simulated RL environment with:
  - States
  - Actions
  - Rewards
  - Agent interaction

### 🤖 LLM-Based Waste Classification
- AI agent analyzes waste input
- Predicts the correct waste category
- Learns through reward-based feedback

### 📊 Structured Logging System
The environment generates organized logs including:
- `START`
- `STEP`
- `END`

This helps track agent behavior, decisions, and performance throughout the simulation.

### ♻️ Smart Waste Segregation
Supports classification of different waste categories such as:
- Plastic
- Metal
- Paper
- Organic Waste
- Other recyclable materials

---

## ⚙️ How It Works

1. The environment generates a waste object.
2. The AI agent receives the waste input.
3. The LLM analyzes and classifies the waste.
4. The environment evaluates the prediction.
5. Rewards or penalties are assigned based on accuracy.
6. Logs are generated for each interaction step.
